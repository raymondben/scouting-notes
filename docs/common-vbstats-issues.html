Volleyball scouting notes

6 Common VBStats issues

  • At the start of every point, check that the correct setter is designated for both teams (the setter has an asterisk by their number in the right-hand panel). If the setter on court changes, you need to change the designated setter using the lineup button (button 9 at the bottom of this page). If a team is using two setters (e.g. back row setters in a 6-2 style rotation system) then you will need to change the designated setter every three rotations.

  • As of VBStats version 2.34.232, it is not possible to enter further actions (during the same point) after a serve ace or attack kills. But if we want to know who made the reception or dig error, we need to enter that action as well. There are three options, with the first being the preferred method:

    • enter the serve as S3 (with its ball path and serve type), then enter the pass action (error), then go back and alter the serve evaluation to “Ace”. This will also work for attack kills/dig errors (enter the attack as “Spike in play” first, then the dig error, then change the spike to “Kill”). This method will work for both serve ace/pass errors as well as attack kill/dig errors.
    • enter only the reception error. The serve action will be automatically entered, but you then have to go back to the serve and enter its ball path and serve type. (This won’t work for attack kill/dig errors though).
    • use the short cut panel for two-team scouting to enter the passing player number and pass error, then go back to the serve and enter the ball path and serve type. (This won’t work for attack kill/dig errors, and also tends not to get the timing correct with respect to the video).

  • If the right-hand tool bar is showing the wrong server (a team’s rotation has gotten out of whack), you can change the server with the lineup button (button 9 at the bottom of this page).

  • If the scores needs adjusting (e.g. if the video has dropped out for a period of time, and you need to re-start scouting when the video comes back on), click the scores in the middle of the top toolbar (button 6) and adjust as appropriate. You might also need to adjust the team rotations with the lineup button (see previous point).

  • To edit the setting zone that has been assigned to an already-entered attack, go to the video screen (the video frame icon at the top right of the match detail screen, which is button 23 here). Find the action to edit, and left-swipe. Click the pencil icon and edit as required. This also allows you to un-select the setting zone (i.e. set it back to “no setting zone”), which you can’t currently do during live scouting.
